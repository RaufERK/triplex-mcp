# Tabs

Компонент навигации первого уровня, используемый для организации и переключения между основными разделами приложения. Визуально и структурно доминирует над другими элементами интерфейса, располагаясь непосредственно под шапкой.

## Особенности

* **Типы**: type1, type2
* **Размеры**: small (SM), medium (MD), large (LG)

| Name | Description                              | Default | Control                    |
| ---- | ---------------------------------------- | ------- | -------------------------- |
| size | Размер компонентаEComponentSizeundefined | md      | Choose option...smmdlg     |
| type | Тип компонентаETabsExtendedTypeundefined | type1   | Choose option...type1type2 |
