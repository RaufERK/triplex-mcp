# ButtonDropdown

Компонент кнопки с выпадающим списком действий.

## Особенности

* **Темы**: General, Secondary, SecondaryLight, Danger
* **Размеры**: small (SM), medium (MD), large (LG)
* **Состояния**: disabled, block
* **Доступность**: ARIA-атрибуты, поддержка клавиатурной навигации

| Name             | Description                                                                                                            | Default | Control |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------- | ------- | ------- |
| theme\*          | Тема кнопки.EButtonTheme.GENERALEButtonTheme.SECONDARYEButtonTheme.SECONDARY\_LIGHTEButtonTheme.DANGEREButtonDotsTheme | \-      | \-      |
| block            | Блочное состояние кнопки.booleanundefined                                                                              | \-      | \-      |
| buttonAttributes | HTML-атрибуты кнопки.ButtonHTMLAttributes<HTMLButtonElement> \| undefined                                              | \-      | \-      |
| size\*           | Размер кнопки.EComponentSize                                                                                           | \-      | \-      |
| options\*        | Список опций.IButtonDropdownOption\[\]                                                                                 | \-      | \-      |
| selected         | Выбранная опция.IButtonDropdownOptionundefined                                                                         | \-      | \-      |
| disabled         | Отключенное состояние кнопки.booleanundefined                                                                          | \-      | \-      |
