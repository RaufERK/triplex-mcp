# OrderedList

Ordered list for displaying sequential items.

**Usage example:**

import { OrderedList } from "@sberbusiness/triplex-next/components/OrderedList";

<OrderedList>
    <OrderedList.Item>List item text;</OrderedList.Item>
    <OrderedList.Item>List item text;</OrderedList.Item>
    <OrderedList.Item>List item text.</OrderedList.Item>
</OrderedList>
