# CheckboxTree

Дерево чекбоксов. Является оберткой над CheckboxTreeExtended.

## Особенности

* **Размеры**: SM, MD, LG.

## Props

| Name         | Description                                                                     | Default           |
| ------------ | ------------------------------------------------------------------------------- | ----------------- |
| checkboxes\* | Набор чекбоксов.ICheckboxTreeCheckboxData\[\]                                   | \-                |
| onChange\*   | Обработчик изменения чекбоксов.(options: ICheckboxTreeCheckboxData\[\]) => void | \-                |
| size         | Размер дерева чекбоксов.EComponentSizeundefined                                 | EComponentSize.MD |
