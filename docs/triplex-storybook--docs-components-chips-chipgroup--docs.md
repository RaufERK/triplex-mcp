# ChipGroup

Контейнер для группировки нескольких компонентов Chip.

## Особенности:

* **Режим отображения**: oneLine (однострочный с горизонтальным скроллом), multiLine (перенос по строкам)
* **Размеры**: sm, md, lg

| Name    | Description                                                | Default           | Control |
| ------- | ---------------------------------------------------------- | ----------------- | ------- |
| oneLine | Чипсы выводятся в одну строку со скроллом.booleanundefined | \-                | \-      |
| size    | Размер компонента.EComponentSizeundefined                  | EComponentSize.MD | \-      |
