# Checkbox

Компонент чекбокса с поддержкой различных состояний и режимов.

## Особенности

* **Размеры**: small (SM), medium (MD), large (LG)
* **Группировка**: по осям X (компонент CheckboxXGroup) и Y (компонент CheckboxYGroup)

## Props

| Name            | Description                                                               | Default |
| --------------- | ------------------------------------------------------------------------- | ------- |
| labelAttributes | Объект label-атрибутов.LabelHTMLAttributes<HTMLLabelElement> \| undefined | \-      |
| bulk            | Признак частичного типа выбора.booleanundefined                           | \-      |
| size            | Размер чекбокса.EComponentSizeundefined                                   | \-      |
