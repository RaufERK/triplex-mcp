# Radio

Радио-кнопка с описанием.

## Особенности

* **Размеры**: small (SM), medium (MD), large (LG)
* **Группировка**: по осям X (компонент RadioXGroup) и Y (компонент RadioYGroup)

## Props

| Name            | Description                                                               | Default |
| --------------- | ------------------------------------------------------------------------- | ------- |
| labelAttributes | Объект label-атрибутов.LabelHTMLAttributes<HTMLLabelElement> \| undefined | \-      |
| size            | Размер радио-кнопки.EComponentSizeundefined                               | \-      |
