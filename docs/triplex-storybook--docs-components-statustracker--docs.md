# StatusTracker

Компонент предназначен для визуального отображения статуса документа.

## Особенности

* **Типы - type**: Draft - черновик, создан без ошибок и еще не отправлен в банк, Waiting - в обработке, ожидает ответ от банка, Warning - предупреждение, документ создан с ошибками, Rejected - документ отклонен банком, Approved - документ исполнен банком
* **Выравнивание по вертикали - verticalAlign**: Top, Middle, Bottom
* Ширину статус-трекера определяет родительский контейнер
