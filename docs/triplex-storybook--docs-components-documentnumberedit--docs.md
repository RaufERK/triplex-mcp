# DocumentNumberEdit

Компонент для редактирования номера документа.

| ------------------------ | ---------------------------------------------------------------------------------------------------------------- | ------- | ------- |
| value                    | Номер документа.ReactTextundefined                                                                               | \-      | \-      |
| buttonLabel\*            | Текст кнопки "Изменить". По умолчанию - "Изменить".string                                                        | \-      | \-      |
| emptyNumberButtonLabel\* | Текст кнопки "Изменить" при отсутствии номера. По умолчанию - "Задать номер".string                              | \-      | \-      |

| numberLabel\*            | Текст перед номером. По умолчанию - "Документ №".string                                                          | \-      | \-      |
