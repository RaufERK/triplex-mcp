# TabsExtended

Базовый компонент табов. На его основе можно рендерить табы любого дизайна.

## Особенности

* Компонент формируется декларативно

| Name          | Description                                     | Default                   | Control |
| ------------- | ----------------------------------------------- | ------------------------- | ------- |
| selectedId\*  | Уникальный идентификатор выбранного таба.string | \-                        | \-      |
| onSelectTab\* | Обработчик выбора таба.TTabsExtendedOnSelectTab | \-                        | \-      |
| type          | Тип компонента.ETabsExtendedTypeundefined       | ETabsExtendedType.TYPE\_1 | \-      |
