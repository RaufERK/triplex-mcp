# Chip

Компонент предоставляет возможность произвести действие по нажатию, также отображает выбранное состояние.

## Особенности:

* **Размеры**: SM (маленький), MD (средний), LG (большой - по умолчанию)
* **Состояния**: selected (выбранное), disabled (отключено)
* Возможно передать префикс и постфикс

| Name     | Description                                                                 | Default           | Control |
| -------- | --------------------------------------------------------------------------- | ----------------- | ------- |
| disabled | Состояние disabled.booleanundefined                                         | \-                | \-      |
| selected | Выбранное состояние.booleanundefined                                        | \-                | \-      |
| prefix   | Контент, предшествующий основному контенту, например иконка слева.ReactNode | \-                | \-      |
| postfix  | Контент, следующий за основным контентом, например иконка справа.ReactNode  | \-                | \-      |
| size     | Размер Chip.EComponentSizeundefined                                         | EComponentSize.MD | \-      |
