# UnorderedList

Unordered list for displaying non-sequential items.

**Usage example:**

import { UnorderedList } from "@sberbusiness/triplex-next/components/UnorderedList";

<UnorderedList>
    <UnorderedList.Item>List item text;</UnorderedList.Item>
    <UnorderedList.Item>List item text;</UnorderedList.Item>
    <UnorderedList.Item>List item text.</UnorderedList.Item>
</UnorderedList>
